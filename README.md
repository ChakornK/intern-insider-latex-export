# Intern Insider LaTeX Resume Exporter

A userscript to export resumes and cover letters generated by Intern Insider in the LaTeX format

---

[![Install](https://img.shields.io/badge/%E2%AD%B3%20Install-003388?style=for-the-badge&cacheSeconds=67)](https://github.com/chakornk/intern-insider-latex-export/releases/latest/download/intern-insider-latex-export.user.js)

Make sure you have [Violentmonkey](https://violentmonkey.github.io) installed on your browser.

## Development

Install Violentmonkey in your browser

Make sure you have corepack installed and enabled

```bash
corepack enable
```

Install dependencies

```bash
yarn
```

Start the development server

```bash
yarn dev
```

The script will be loaded into Violentmonkey. Enable <kbd>Track external edits</kbd>.
