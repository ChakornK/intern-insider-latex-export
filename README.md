# Intern Insider Enhancements

A userscript to add QoL features to Intern Insider.

Current features:
- Export generated resumes and cover letters in the LaTeX format

---

[![Install](https://img.shields.io/badge/%E2%AD%B3%20Install-003388?style=for-the-badge&cacheSeconds=67)](https://github.com/chakornk/intern-insider-enhancements/releases/latest/download/intern-insider-enhancements.user.js)

Make sure you have [Violentmonkey](https://violentmonkey.github.io) installed on your browser.

## Development

Install Violentmonkey in your browser

Make sure you have corepack installed and enabled

```bash
corepack enable
```

Install dependencies

```bash
yarn
```

Start the development server

```bash
yarn dev
```

The script will be loaded into Violentmonkey. Enable <kbd>Track external edits</kbd>.
